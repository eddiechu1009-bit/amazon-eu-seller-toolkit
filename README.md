# Amazon 歐洲賣家營運工具箱

註冊完成後的六大必修課，從品牌註冊到廣告投放一站搞定。

## 功能模組

| 模組 | 說明 | 項目數 |
|------|------|--------|
| 🏷️ 品牌註冊 | Brand Registry 申請流程與商標準備 | 5 |
| 📦 商品上架前置 | 關鍵字研究、圖片、文案、A+ Content、定價 | 10 |
| 🚚 建立 Shipment | 條碼、標籤、箱規、報關、貨代選擇 | 8 |
| 🚀 開賣後必做五件事 | Vine、自動定價、報表監控、庫存、客服 | 6 |
| 📢 廣告策略 | PPC 五階段路線圖與預算建議 | 7 |
| 🎯 促銷工具 | Coupons、Deals、Prime Discount、促銷日曆 | 8 |

## 特色

- ✅ Checklist 打勾追蹤進度（localStorage 持久化）
- 📊 整體 / 模組進度條
- 📱 響應式設計，手機可用
- 🚀 純前端靜態網站，無需後端

## 開發

```bash
npm install
npm run dev
```

## 部署

支援 Netlify、Vercel、GitHub Pages 等靜態託管：

```bash
npm run build
```

產出的 `dist/` 資料夾即可直接部署。

## 技術棧

- React 18 + TypeScript
- Vite
- Tailwind CSS
